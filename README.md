# Fortune 2025 (포춘 2025)

당신의 2025년 운세는 어떨까요? 재미로 보는 신년운세 테스트입니다.

## 🌟 특징
- 12가지 질문으로 알아보는 2025년 운세
- 6가지 상세한 운세 결과 제공
- 직관적인 UI/UX
- 모바일 최적화 디자인

## 🎯 결과 유형
- 황금빛 성장운
- 달콤한 인연운
- 럭키 머니운
- 창의적 도전운
- 안정적 성취운
- 긍정적 변화운

## 💻 사용 기술
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 🎨 디자인 특징
- 모던한 UI/UX
- 반응형 웹 디자인
- 고급스러운 시각적 요소
- 직관적인 사용자 경험
